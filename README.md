EyeMove for Laravel.

## Install

Via Composer

``` bash
$ composer require lamalama/laravel-eyemove
```

You can optionally publish the config file with:
```bash
php artisan vendor:publish --provider="LamaLama\EyeMove\EyeMoveServiceProvider" --tag="config"
```
